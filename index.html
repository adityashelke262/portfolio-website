Dormify: No-Broker Accommodation Platform
Dormify is a smart hostel and PG finding platform developed to simplify the accommodation search process for students. It provides a centralized system where users can explore, compare, and select suitable living options based on location, price, amenities, and user reviews. The platform eliminates the need for brokers, reducing extra costs and ensuring transparency. It features advanced search and filtering, detailed listings, and nearby facility information such as hospitals, gyms, and transport. With a user-friendly interface and real-time data, it helps students make informed decisions quickly and efficiently. Overall, Dormify enhances convenience, saves time, and offers a reliable solution for finding affordable and comfortable student accommodation.
Automated Street Light Control using Arduino
The Automated Street Light Control System using Arduino is a smart solution designed to reduce energy consumption and improve efficiency in urban lighting. It uses a photoresistor (LDR) to detect day and night conditions and a motion sensor to identify movement. The lights turn on only in darkness and brighten when motion is detected, ensuring optimal usage. This system overcomes the limitations of traditional street lighting, such as fixed schedules, energy wastage, and light pollution. By using simple and cost-effective components like Arduino, sensors, and LEDs, it provides an affordable and scalable solution. Overall, it helps reduce electricity costs, lower carbon emissions, and supports the development of smart and sustainable cities.

AI-Based Farmer Query Support & Advisory System
The AI-Based Farmer Query Support and Advisory System is a smart digital platform designed to help farmers make better agricultural decisions using real-time data and AI technologies. It enables farmers to detect crop diseases through image recognition, receive personalized recommendations, and access accurate weather forecasts for improved planning. The system also provides market price insights and connects farmers directly with buyers, reducing reliance on intermediaries. With multilingual support and a simple, user-friendly interface, it ensures accessibility for farmers with varying levels of digital literacy. By promoting efficient resource use and sustainable practices, this solution enhances productivity, increases income, and supports the digital transformation of agriculture in a scalable and cost-effective way.
